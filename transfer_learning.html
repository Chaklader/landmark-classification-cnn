Question: Outline the steps you took to get to your final CNN architecture and your reasoning at each step. Describe why you think the architecture is suitable for the current problem.

Answer: I decided to use ResNet18 for the base of my model, since it performs fairly well on ImageNet and is not too large of a model. Also, since ResNet18 was trained for the ImageNet task, it is a good model to use for this landmark classificaiton task, since both ImageNet and this landmark task use images of natural scenes.

from src.predictor import Predictor
from src.helpers import compute_mean_and_std
from src.transfer import get_model_transfer_learning

# Get data loaders and class names
data_loaders = get_data_loaders(batch_size=1)
class_names = data_loaders["train"].dataset.classes

# Create new model instance
model_transfer = get_model_transfer_learning("resnet18", n_classes=50)
model_transfer = model_transfer.cpu()

# Load the checkpoint and extract model state dict
checkpoint = torch.load("checkpoints/model_transfer.pt", map_location="cpu")
model_transfer.load_state_dict(checkpoint['model_state_dict'])  # Note this change
model_transfer.eval()

# Wrap with predictor class
mean, std = compute_mean_and_std()
predictor = Predictor(model_transfer, class_names, mean, std).cpu()

# Export using torch.jit.script
scripted_predictor = torch.jit.script(predictor)
scripted_predictor.save("checkpoints/transfer_exported.pt")
